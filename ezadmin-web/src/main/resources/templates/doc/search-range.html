区间查询

区间查询默认是 大于等于或小于等于。 适用于数字区间与日期区间

日期区间

参数说明

配置项 说明 默认值
jdbctype 数字靠右展示，时间格式化，小数位保留 DATETIME("DATETIME"),
oper sql操作符，不同的操作符，拼接出来的SQL不一样 BETWEEN("BETWEEN","区间"),
alias 表别名，当多个表字段一样时，用别名来区分，同时如果是需要用表达式替换 如： ifnull(a.id,0) ,最终的SQL也会使用alias来替换字段名 EQ("=","等于"), NE("!=","不等于"), LTE( "<=","小于等于"), BETWEEN("BETWEEN","区间"), GTE(">=", "大于等于"), LIKE("LIKE","包含"), NOTLIKE("NOT LIKE","不包含"), IN("IN","IN"), NOT_IN("NOT IN","NOT IN"), ALL_FIND_IN_SET("ALL_FIND_IN_SET","ALL_FIND_IN_SET");
