下拉框

系统内置了多种下拉框，满足大部分业务场景,同时支持html原生的属性。

公共配置

原生下拉框

参数说明

配置项 说明 默认值
datatype 下拉框的数据来源 TEXT,JSON,KVSQL,HTTP,JOIN,KVSQLCACHEFOREVER,KVSQLCACHE
,HTTPCACHE,QLEXPRESS,KVSQLCACHE5 ,EZLIST,KVJSONRESULT,CASCADER,KVSQL2TREECACHE,DATAGROUP
data 对应datatype的数据取值参数 yesno(是否),status(禁用启用),delete(已删除未删除),数据字典类型,数据源编码
sql select K,V from table where id=${xx} 或者 url
换行符分割
k:v\n 分割
[{"V":"男","K":"1"},{"V":"女","K":"0"}]
如果是JSON， 键必须是 大写 K V ;
col 栅栏，当前字段的宽度 1到12
alias 表的别名
item_desc 下方的注释
top_desc 上方的注释，主要用于富文本
jdbctype 数据库字段类型 NUMBER, NUMBERNULL, DATE_TO_NUMBER, VARCHAR, DATE, DATETIME, CLOB , BODY, NUMBER0, NUMBER1, NUMBER2 , NUMBER3, NUMBER4
lay_verify layui原生验证 required|phone|number 图片：{"required":true,"acceptMime":"image/jpeg, image/jpg","exts":"jpg|jpeg","max":1,"min":1,"size":200,"width":680,"height":300} 文件：{"required":true,"acceptMime":"image/jpeg, image/jpg","exts":"jpg|jpeg","max":1,"min":1,"size":200,"width":680,"height":300} 参考layui官网
valid_rule jQuery 验证规则 {"required": true, "minlength": 2}
valid_msg 下方的注释 {"required": true, "minlength": "长度{0}"}
help 字段名称上面的提示，一般用不到
其他 参见HTML原生属性
