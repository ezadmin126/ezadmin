静态表格

服务端生成静态的表格，由前端根据样式渲染成需要的展现，主体配置分别通过body,自定义元素表示。

示例

参数说明

配置项 说明 默认值
datasource 数据源 datasource
empty_show 列表全局为空或者null时展示的字符串 空格
tablestyle 列表是否为原始html pure: 原始html，主要用于一些 纯css渲染table的情况
空或者其他（默认）： layui渲染
adminstyle 列表样式 内置为layui
pagesync 分页是否同步加载 空或其他：异步加载 1：同步加载
export 工具栏是否包含导出按钮 空或其他：不包含 1：包含
cellminwidth 列宽最小值 110
linestyle 行高等 linestyle="height:100px"
expandedMode 展开模式 tips 悬浮展开方式 default 多行展开方式（默认）
