开始使用

EZ是一个嵌入式的Servlet表单列表工具，可以将SQL转换为列表（table）或表单（form），可适配各类UI框架，同时支持拖拽生成列表表单。

1.maven依赖（内置springboot-starter）

jdk1.8+
新建一个空的springboot应用 ，版本建议：2.2.6.RELEASE

添加maven依赖


  io.github.ezadmin126
  ezadmin-core
  2.9.14.7

可能产生jar冲突的依赖：
jsqlparser4.2
caffeine:2.9.3
QLExpress:3.3.1
jsoup:1.16.1
