参数说明

配置项 说明 默认值
windowname 弹框标题 新增
url 弹框或跳转的url
area 窗口的大小 100 或者 300,100
class 边框颜色 layui-border-blue
display 是否展示 display="spring("orderService").exe("1")" ql表达式，返回0表示不展示，默认展示
opentaype 打开方式
  • MODEL：弹框
  • SCRIPT：自定义事件
  • CONFIRM_MODEL：带确认的弹窗
  • _BLANK：新浏览器窗口
  • LOCATION：当前窗口
  • PARENT：父窗口打开标签
  • APPEND_PARAM：携带当前搜索参数窗口打开
  • _BLANK_PARAM：携带当前搜索参数新浏览器窗口
  • _BLANK_PARAM_COLUMN：只携带配置显示的参数，用于导出
  • AJAX：异步请求
  • CONFIRM_AJAX：带确认的异步请求
  • FORM：全屏表单
