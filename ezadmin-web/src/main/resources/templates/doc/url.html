URL约定

本着约定大于配置的原则，EZ自身实现了一些业务场景。

系统

配置项 说明 默认值
/topezadmin/index.html 管理系统首页URL
/topezadmin/navs.html 管理系统默认菜单URL
/topezadmin/dark.html 管理系统 切换为黑夜模式

列表

配置项 说明 默认值
/topezadmin/list/list-${CODE} 生产模式列表查询
/topezadmin/list/export-${CODE} 生产模式列表导出 导出按钮，或者设置主体export=1.右侧工具栏导出
/topezadmin/listEdit/list-${CODE} 开发模式列表查询
/topezadmin/listEdit/sourceEdit-${CODE} 开发模式列表源码编辑
/topezadmin/listEdit/loadEdit-${CODE} 开发模式列表图形编辑
/topezadmin/list/count-${CODE} 列表异步获取分页
/topezadmin/list/tree-${CODE} 树形列表
/topezadmin/list/treedata-${CODE} 树形列表异步获取数据
/topezadmin/list/trace-${CODE} 打印执行日志 查看执行的所有SQL
/topezadmin/list/api-${CODE} 数据以json格式返回，适用于前后端分离等场景 满足EzResult格式
/topezadmin/list/selectCols-${CODE} 设置可见列
/topezadmin/list/doOrder-${CODE} 执行list里面配置的排序表达式 参见

表单

配置项 说明 默认值
/topezadmin/form/form-${CODE} 生产模式表单
/topezadmin/form/doSubmit-${CODE} 生产模式表单提交
/topezadmin/form/doStatus-${CODE} 生产模式表单状态修改
/topezadmin/form/doDelete-${CODE} 生产模式表单删除
/topezadmin/form/trace-${CODE} 生产模式表单SQL查看 在url后面拼一个 debug=1 参数也可行
/topezadmin/form/upload.html 自带的文件上传
/topezadmin/form/download.html 自带的文件下载
/topezadmin/form/detail-${CODE} 详情页，布局跟form的样式一直
