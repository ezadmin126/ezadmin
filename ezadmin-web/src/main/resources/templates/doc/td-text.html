文本

展示数据库查出来的数据

普通文本

默认插件，直接展示文本，会去除html标签，类似thymeleaf的 th:text

配置项 说明 默认值
width 列宽 110
datatype 数字靠右展示，时间格式化，小数位保留 NUMBER, NUMBERNULL, DATE_TO_NUMBER, VARCHAR, DATE, DATETIME, CLOB , BODY, NUMBER0, NUMBER1, NUMBER2 , NUMBER3, NUMBER4 VARCHAR

链接文本

默认插件，直接展示文本，会去除html标签，类似thymeleaf的 th:text

配置项 说明 默认值
width 列宽 110
url 弹框或跳转的url

窗口链接

默认插件，直接展示文本，会去除html标签，类似thymeleaf的 th:text

配置项 说明 默认值
width 列宽 110
windowname 弹框标题 新增
url 弹框或跳转的url
area 窗口的大小 100 或者 300,100
opentaype 打开方式
  • MODEL：弹框
  • SCRIPT：自定义事件
  • CONFIRM_MODEL：带确认的弹窗
  • _BLANK：新浏览器窗口
  • LOCATION：当前窗口
  • PARENT：父窗口打开标签
  • APPEND_PARAM：携带当前搜索参数窗口打开
  • _BLANK_PARAM：携带当前搜索参数新浏览器窗口
  • _BLANK_PARAM_COLUMN：只携带配置显示的参数，用于导出
  • AJAX：异步请求
  • CONFIRM_AJAX：带确认的异步请求
  • FORM：全屏表单
