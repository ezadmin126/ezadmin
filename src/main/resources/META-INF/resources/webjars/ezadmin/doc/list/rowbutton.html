每个数据行上的按钮 
<th type="rowbutton" id="rowbutton"><button class="layui-btn list-row-button" type="group" opentype="_BLANK" url="/ezadmin/list/listEdit-${ENCRYPT_LIST_ID}" windowname="修改${LIST_NAME}" name="update">修改 </button></th>
字段名称>name
用于自定义按钮事件，区分各个按钮
文案名称>text()
展示的文案 <button>新增</button>
跳转链接>url
相对或绝对的地址
打开方式>opentype
MODEL("1","弹窗"),
_BLANK("2","新浏览器窗口"),
LOCATION("3","当前窗口"),
PARENT("4","父窗口打开标签"),
APPEND_PARAM("5","携带当前搜索参数窗口打开"),
_BLANK_PARAM("6","携带当前搜索参数新浏览器窗口"),
窗口名称>windowname
修改${USER_NAME} 支持获取request参数
窗口大小>area
1.如果选择弹框，可以定义窗口大小。默认 ['90%', '90%']
2.可以餐卡layer配置jsonarray
3.逗号分割： 100px,200px
4.高度自适应：100px 或 90%
自定义样式>style
设置操作列的样式，只有第一个按钮生效，默认：max-width:175px
插件>type
1.平铺(建议)：single
2.分组：group
3.单选：singleselectradio
回调函数：
ez_callback||url
ez_callback为请求URL的参数，如果为空，则取：跳转链接>url
ez_callback(ids,lines);
ids为选中的ID，lines为当前行的所有数据
是否展示表达式>display
 当前这行数据的  ID
是否展示>display
    按钮根据这行的数据控制是否展示
