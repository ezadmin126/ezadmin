表单名称>title

数据源>body>datasource
ezadmin.datasourceBeanNames配置的数据源列表

提交表达式>formitem>SUBMIT_EXPRESS

添加了一些自定义类，便于增减字段。
insertSimple会根据InsertParam的值，生成SQL，最终也是执行 insert(sql)或者update(sql);
特殊情况可以：调用springbean ，传入request参数Map，执行特殊业务逻辑。
当然也可以自定义函数。
id= spring("productService").create($("request"));
import com.ezadmin.plugins.express.jdbc.UpdateParam;
import com.ezadmin.plugins.express.jdbc.InsertParam;
companyId=$$("COMPANY_ID");
sessionUserId=$$("EZ_SESSION_USER_ID_KEY");
sessionUserName=$$("EZ_SESSION_USER_NAME_KEY");
if(!isNotBlank("ID")){
    param=new InsertParam();
    param.table("T_BASE_ATTRIBUTE");
     param.add("#{ATTRIBUTE_NAME}");
    param.add("#{COMPANY_ID,value="+companyId+"}");
    param.add("#{ADD_TIME,value=NOW()}");
    param.add("#{ADD_ID,value='"+sessionUserId+"'}");
    param.add("#{ADD_NAME,value='"+sessionUserName+"'}");
    id=insertSimple(param);
    return id;
}else{
    param=new UpdateParam();
    param.table("T_BASE_ATTRIBUTE");
    param.add("#{ATTRIBUTE_NAME}");
    param.add("#{UPDATE_TIME,value=NOW()}");
    param.add("#{UPDATE_ID,value='"+sessionUserId+"'}");
    param.add("#{UPDATE_NAME,value='"+sessionUserName+"'}");
    StringBuilder updateSql=new StringBuilder();
    updateSql.append(" where ATTRIBUTE_ID=#{ID} ");
    if(companyId>1){
        updateSql.append(" and COMPANY_ID= "+companyId);
    }
    param.where(updateSql.toString());
    updateSimple(param);
    return ID;
}

自定义成功URL>body>SUCCESS_URL
保持成功之后跳转的url,默认编辑页。 reload:全局刷新，主要是 iframe格式的页面
reloadlocal: 仅限跳转当前页面，刷新当前tab
http:xxx:跳转到其他页面,配置绝对路径或者相对路径，url支持${xxx}获取request或者返回值里面的数据
