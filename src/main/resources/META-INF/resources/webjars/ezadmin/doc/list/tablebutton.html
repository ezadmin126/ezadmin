table左上角的按钮，主要是 新增，导出等与行交互较少的功能。 通过配置id为tableButton的节点来实现。 
   <div class="btn-group   bd-highlight" id="tableButton">
          <button item_name="新增" name="新增" url="/ezadmin/form/form-base_regist" opentype="MODEL" windowname style type="table">
          新增
          </button>
   </div>
字段名称>name
字段名称，如果想通过js操作次按钮，会有点作用
<button name=abc >新增</button>
文案名称>text()
<button>新增</button>
跳转链接>url
相对或绝对的地址
打开方式>opentype 
MODEL("1","弹窗"),
_BLANK("2","新浏览器窗口"),
LOCATION("3","当前窗口"),
PARENT("4","父窗口打开标签"),
APPEND_PARAM("5","携带当前搜索参数窗口打开"),
_BLANK_PARAM("6","携带当前搜索参数新浏览器窗口"),
窗口名称>windowname
修改${USER_NAME} 支持获取request参数
窗口大小>area
1.如果选择弹框，可以定义窗口大小。默认 ['90%', '90%']
2.可以餐卡layer配置jsonarray
3.逗号分割： 100px,200px
4.高度自适应：100px 或 90%
插件>type
1.普通：table
2.弹框多选按钮：tableselectcheckbox
3.弹框单选按钮：tableselectradio
回调函数：
ez_callback||url
ez_callback为请求URL的参数，如果为空，则取：跳转链接>url

ez_callback(ids,lines);
ids为选中的ID，lines为当前行的所有数据
