EZadmin 是一款低代码后端框架，它使用 HTML+JS+CSS 来配置生成页面，可以节省页面开发工作量，极大提升开发前端页面的效率；

能快速将一条 SQL 语句转换成：

普通列表，excel 导出，表单，API 接口，树形列表。

也能导出静态html页面

1.加入jar包

```xml
<dependencies>
  <dependency>
	<groupId>top.ezadmin</groupId>
	<artifactId>ezadmin-core</artifactId>
	<version>1.0.0-SNAPSHOT</version>
  </dependency>
</dependencies>
```
