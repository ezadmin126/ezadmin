✨ 零配置

只需引入 JS 文件，无需调用任何初始化方法。

🎨 Layui 集成

使用 Layui Layer 弹框和表单组件，风格统一。

🔍 智能选择

只选中带有 item_name 属性的元素，避免误选。

💡 简化样式

选中状态使用简单边框，不影响页面布局。

💻 代码示例

实现这个页面只需要：

<!-- 1. 引入 Layui -->
<link rel="stylesheet" href="layui.css">
<script src="layui.js"></script>
<script src="jquery.js"></script>

<!-- 2. 引入 EZ Inspector - 就这么简单！-->
<link rel="stylesheet" href="ez-inspector.css">
<script src="ez-inspector.js"></script>

<!-- 3. 给需要检查的元素添加 item_name 属性 -->
<div item_name="my_element">这个元素可以被选中</div>

📋 使用说明

  1. 页面加载后，Inspector 自动激活，检查面板会自动弹出
  2. 鼠标悬停在带有 item_name 的元素上会显示蓝色高亮框
  3. 点击元素，会在元素上显示绿色边框，面板中显示元素信息
  4. 在面板的备注框中输入说明，点击"提交"按钮（数据会输出到控制台）
  5. 打开浏览器控制台（F12）查看提交的数据
  6. 按 Ctrl + Shift + I 可以切换检查模式的激活/停用
