# EzAdmin 低代码配置工具使用说明

## 一、概述

EzAdmin 是一个基于 JSON 配置的 列表表单 生成工具。它使用 Thymeleaf 作为模板引擎，支持 QL 表达式进行数据查询，并且大部分配置兼容 Layui 的原生配置。

## 二、快速开始

### 2.1 Maven 依赖

**Java 17:**
```xml
<dependency>
    <groupId>io.github.ezadmin126</groupId>
    <artifactId>ezadmin-java17-spring-starter</artifactId>
    <version>3.0.6</version>
</dependency>
```

**Java 8:**
```xml
<dependency>
    <groupId>io.github.ezadmin126</groupId>
    <artifactId>ezadmin-java8-spring-starter</artifactId>
    <version>3.0.6</version>
</dependency>
```

### 2.2 启用 EzAdmin

```java
@SpringBootApplication
@EnableEzadmin
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

## 五、表达式系统

EzAdmin 使用 QL Express 作为表达式引擎，支持 SQL 查询和数据处理。

其中内置了如下函数：
 
1. select 查询sql
2. selectOne 查询sql，返回第一条数据
3. search 支持按照搜索参数，自动拼接where语句的搜索查询
4. treeSearch 持按照搜索参数，自动拼接where语句的树形搜索查询
5. count 查询sql，获取总数
6. $  获取请求参数 $("xxx")
6. $$  获取请求session里面的参数 $￥("xxx")
7. isNotBlank request里面的参数是否不为空
8. isBlank request里面的参数是否为空
 
9. update  执行update sql语句
10. insert 执行insert sql语句
11. insertSimple  对象参数的格式，执行insert操作，如：
    COMPANY_ID=1;
    param=new InsertParam();
    param.table("T_SYS_ORGANIZATION");
    param.add("#{COMPANY_ID,value="+COMPANY_ID+"}");
    param.add("#{ADD_TIME,value=now()}");
    param.add("#{ADD_ID,value="+EZ_SESSION_USER_ID_KEY+"}"); 
    param.add("#{DELETE_FLAG,value=0}");
    id=insertSimple(param);
12. updateSimple

EzAdmin 大部分配置兼容 Layui 原生配置。
 

 

### 6.3 props 中的 Layui 属性

在组件的 props 中，可以直接使用 Layui 的原生属性：

```json
{
    "props": {
        "lay-verify": "required",
        "lay-verType": "tips",
        "lay-search": true,
        "lay-filter": "myFilter"
    }
}
```

## 九、API 参考

### 9.1 URL 路由

| URL | 说明 |
|-----|------|
| `/topezadmin/list/list-{code}` | 列表页面 |
| `/topezadmin/list/data-{code}` | 列表数据接口 |
| `/topezadmin/list/count-{code}` | 列表总数接口 |
| `/topezadmin/list/tree-{code}` | 树形列表页面 |
| `/topezadmin/form/form-{code}` | 表单页面 |
| `/topezadmin/form/save-{code}` | 表单保存接口 |
| `/topezadmin/export/export-{code}` | 数据导出 |

### 9.2 数据格式

**列表数据响应：**
```json
{
    "code": 0,
    "msg": "success",
    "count": 100,
    "data": [
        {"id": 1, "username": "admin"},
        {"id": 2, "username": "user"}
    ]
}
```

**表单保存响应：**
```json
{
    "code": 0,
    "msg": "保存成功"
}
```
